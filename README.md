### 1. Clone repository

```bash
git clone <repository-url>
cd alpha-test
```

### 2. Install dependencies

```bash
npm install
```

or using yarn:

```bash
yarn install
```

### 3. Run the project in development mode

```bash
npm run dev
```

or:

```bash
yarn dev
```


or:

```bash
yarn build
```

